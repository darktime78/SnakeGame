## Snake game in Terminal

### Compile:
    Need GCC or Clang installed
    
    gcc -Wall -pedantic snake-game.c -o snake-game
    clang -Wall -pedantic snake-game.c -o snake-game

### Run:
    ./snake-game

### Control:
    Up,down,left,right arrows - snake movement direction
    Escape or 'q' - quit
